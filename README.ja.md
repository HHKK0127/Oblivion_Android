Android NDK と OpenGL ES 3.0 を使用して、C++ で完全に実装された The Elder Scrolls IV: Oblivion の Android ネイティブポート。

## 🎮 機能

### 実装されたコアシステム
- ✅ **3D レンダリングエンジン** - OpenGL ES 3.0 によるメッシュとテクスチャサポート
- ✅ **ゲーム世界** - セルベースの世界システムとシームレスな遷移
- ✅ **NPC システム** - 100+ の NPC と AI ステートマシン（アイドル、徘徊、巡回、戦闘、追従）
- ✅ **戦闘システム** - ステータスと装備を考慮した完全なダメージ計算
- ✅ **クエストシステム** - マルチオブジェクティブクエストと報酬（金、経験値）
- ✅ **魔法システム** - 6 つの学派と 10+ の呪文、マナ管理
- ✅ **キャラクタステータス** - HP、マナ、スタミナ、属性、スキル
- ✅ **ローカライゼーション** - 日本語 + 英語（100+ 翻訳）
- ✅ **パフォーマンスモニタリング** - フレームタイミング、メモリ、CPU プロファイリング
- ✅ **テキストレンダリング** - スクリーン上のテキスト表示（色、位置指定）
- ✅ **デバッグ HUD** - FPS、フレームタイム、メモリ、システム情報オーバーレイ
- ✅ **設定システム** - 永続的なデバッグモードと言語設定
- ✅ **セーブ/ロードシステム** - ゲーム状態の永続化とスロット管理
- ✅ **OpenAL 3D オーディオ** - 距離減衰による空間オーディオ
- ✅ **レトロフィルターエフェクト** - ピクセル化、スキャンライン、色削減、CRT 歪み、フィルムグレイン

## 📦 ビルド＆インストール

### 前提条件
```bash
# Android SDK/NDK をインストール
sdkmanager "ndk;26.1.10909125"
sdkmanager "cmake;3.16.0"

# リポジトリをクローン
git clone https://github.com/HHKK0127/Oblivion_Android.git
cd oblivion-android
```

### リリース APK をビルド
```bash
# ビルドと署名
./gradlew clean assembleRelease

# 出力
# 場所: app/build/outputs/apk/release/app-release.apk
# サイズ: ~8 MB
```

### デバイスにインストール
```bash
# ADB 経由
adb install -r app/build/outputs/apk/release/app-release.apk

# またはデバイスに APK をコピーしてインストール
```

## 🚀 はじめに

1. **アプリを起動**: ホーム画面の Oblivion アイコンをタップ
2. **タイトルスクリーン**: 3 秒待機してからタップして開始
3. **メインゲーム**: Oblivion の世界を探索
4. **NPC との相互作用**: 近くのキャラクターをタップ
5. **戦闘**: 敵との戦闘が自動的に始まります
6. **クエスト**: NPC ダイアログから受け入れます
7. **魔法**: 戦闘中に呪文をキャスト
8. **ログを確認**: クエスト進捗を表示

### ゲームコントローラー
- **周りを見る**: 画面をドラッグしてカメラを回転
- **相互作用**: NPC またはオブジェクトをタップ
- **メニュー**: クエスト UI に現在のクエストを表示
- **魔法**: NPC は戦闘中に自動キャスト（将来：手動キャスト）
- **設定**: タイトルメニューの「設定」をタップしてアクセス

## 🎨 UI とデバッグシステム

### 設定メニュー
タイトルスクリーンからアクセス：
1. **タイトルスクリーン** → 「設定」をタップ
2. **設定パネル** が 3 つのオプションで表示：
   - **デバッグモード**: ON/OFF を切り替えてデバッグ HUD を表示/非表示
   - **言語**: 日本語と英語を切り替え
   - **戻る**: メインメニューに戻る

設定は自動的に永続ストレージに保存されます。

### デバッグ HUD 表示（Phase 8 拡張）
**デバッグモード: ON** のとき、リアルタイムで表示：
- **FPS**: 現在のフレーム数（例：「FPS: 60.0」）
- **フレームタイム**: フレームごとのミリ秒（例：「Frame: 16.67 ms」）
- **平均**: 実行中の平均フレームタイム（例：「Avg: 16.50 ms」）
- **メモリ**: 現在の RAM 使用量（例：「Mem: 45 MB」）
- **キューブ**: アクティブなゲームオブジェクト数
- **ステータス**: 「DEBUG: ON/OFF」を表示
- **オーディオシステム**（Phase 8）: ロード済みクリップ、アクティブなソース、BGM ステータス（例：「Audio: clips=5 sources=12」）
- **レトロフィルター**（Phase 8）: アクティブなエフェクト略称（例：「Filters: SPG」= スキャンライン、ピクセル化、グレイン）

### テキストレンダリングシステム
- OpenGL ES 3.0 正射影を使用
- 任意のスクリーン座標での色付きテキストをサポート
- ネイティブ解像度でのスケーリングでレンダリング
- すべての UI システム（タイトル、クエスト、設定）に統合
